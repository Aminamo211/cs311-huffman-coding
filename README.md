# Huffman Coding Project 

## Purpose
This project uses Huffman Coding to encode and decode text files using binary codes.

## Data Structures Used
- Priority Queue (Min Heap)
- Binary Tree
- Hash Map

## Files
- main.cpp
- input.txt
- encoded.txt
- decoded.txt

## How to Compile

```bash
g++ main.cpp -o huffman
```

## How to Run

```bash
./huffman
```

## Program Output
The program creates:
- encoded.txt
- decoded.txt

## Description
The program reads text from input.txt, counts character frequencies, builds a Huffman tree, encodes the text into binary, and decodes it back into the original text.
